INTRODUCCIÓN REGRESIÓN LOGÍSTICA

La regresión logística es una técnica de análisis de datos para hallar la probabilidad de un evento. Ibm.com la explica en acotaciones de 0 a 1 dependiendo de cada caso y teniendo en cuenta la fórmula de probabilidad (Ibm.com, 2025). Amazon.com lo explica como el uso de matemáticas entre dos factores de datos para predicción de resultados certeros de forma rápida y sencilla con tratamiento previo de los datos (Amazon.com, S/f). Este tipo de aprendizaje comprende tipos tales como la logística binaria la cual según GeeksforGeeks se basa en “es un tipo de análisis de regresión que se utiliza cuando la variable dependiente es binaria.” (GeeksforGeeks, 2024). Es decir un análisis el cual se basa en dos posibilidades en la mayoría de casos sí o no, debido a su análisis de categorías de forma individual.

Esta regresión también se puede usar para obtener una probabilidad por medio de la función sigmoide, la página de desarrolladores de Google la describe como parte de las funciones logísticas, siendo la función estándar de este campo (Google Developers, 2025). Dicha función consiste en una gráfica con forma de S que con el aumento de x, el valor se aproxima a 1 u 0, más no llegan a tener estos valores exactos, su uso también comprende las redes neuronales para convertir salidas de neuronas en rangos para filtros de paso por probabilidad.

MODELO PARA VARIABLE OBJETIVO BINARIA

En este contexto, MedCalc (s.f.) señala que, “La regresión logística es un método estadístico para analizar un conjunto de datos con una o más variables independientes que determinan un resultado”, lo que habla de su principal uso que es el de aplicarse en situaciones donde la variable objetivo es binaria, es decir, cuando solo existen dos posibles resultados como: Si/No, Éxito/Fracaso o Positivo/Negativo. Por lo que podríamos decir que la diferencia entre la regresión logística y la regresión lineal es que la primera, se ocupa de predecir valores que son continuos, mientras que el otro, categoriza eventos en grupos de dos.

PROBABILIDAD MEDIANTE SIGMOIDE/LOGIT

Para determinar la probabilidad de que ocurra el evento, este tipo de regresión utiliza la función sigmoide o logit, la cual convierte cualquier numero en un resultado que se encuentra entre 0 y 1, lo que permite que lo interpretemos como una probabilidad. DataCamp (s.f) menciona “la función sigmoidea… da una curva en forma de ‘S’ que puede tomar cualquier número de valor real y convertirlo en un valor entre 0 y 1” lo que nos habla de que esta función es capaz de representar las situaciones que sean de naturaleza binaria, como bien describimos anteriormente, además de facilitar la toma de decisiones en base a datos.
